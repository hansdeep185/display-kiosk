Cara Menggunakan Layanan Print

1. Scan

QR Code di halaman utama atau kirim pesan ke nomor WhatsApp IT Support
2. Kirim

dokumen siap cetak (disarankan format .pdf)
3. Validasi

Setelah semua file terkirim, balas dengan kata "SELESAI"
4. Printing

Untuk memulai cetak, balas pesan konfirmasi dengan "YA"
5. Tunggu

Notifikasi bahwa dokumen Anda Selesai Dicetak !
